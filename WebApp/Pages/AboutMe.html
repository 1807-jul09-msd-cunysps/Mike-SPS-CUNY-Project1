Interesting Projects

Virtual Network

    Setup and configured a custom Mininet environment on Linux using its Python API. Mininet was used to create a virtual network and customize the network topology, giving control over the individual switches, connections, and nodes.

  • Simulated cellular connections by adjusting bandwidth, delay, loss rates, etc. to find their effects on round-trip-time and total packets sent. The goal of this project was to quantify the effect that a client-node’s poor connection has on the server due to the re-transmission of lost packets.
  • Wrote server and client programs to simulate taxis traveling around a city, gathering data, and forwarding the collected data to the server. The goal of this project was to determine how many taxis were needed to maintain a constant flow of data from all sectors of the city.

Stock Analysis

    Developed software for gathering stock-trading data, using Quandl’s API, and relevant news articles, from Google Finance and Seeking Alpha. Data and news-article gathering was automated using Java, running on a GCP VM. News articles were parsed, organized, and filtered using a combination of Java regexes and Hadoop MapReduce prior to analysis. Analysis focused on the relationship between the quantity of news articles, word usage in news articles, and significant stock movements.
